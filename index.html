Coldbrew

The missing part of the missing package manager for macOS

Install Coldbrew

/bin/bash -c "$(curl -fsSL https://coldbrew.ga/install.sh)"

Paste that in a macOS Terminal (we don't support Linux yet).

What does Coldbrew do

Coldbrew fills in the blanks of what homebrew is lacking. We use shell scripts to fix some functionality of brew, and in general just make brew easier to use
